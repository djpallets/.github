# Benvenuto in Pallets! 👋

**Pallets** è un'alternativa moderna al Django Admin, progettata per offrire un'interfaccia più intuitiva, un design contemporaneo e un'esperienza migliore per sviluppatori e staff.

Il progetto si integra facilmente con Django, mantenendo tutta la potenza dell'Admin originale ma con un'esperienza utente completamente rinnovata.

## Repository principali

### Community Edition

La versione gratuita e open source di Pallets.

- ✅ Gratuita
- ✅ Open Source
- ✅ Pensata per sviluppatori, progetti personali e piccoli team

➡️ Repository: https://github.com/djpallets/community
